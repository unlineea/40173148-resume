# تمرین ساخت رزومه با HTML

این پروژه شامل یک فایل ساده‌ی HTML به نام `index.html` است که رزومه‌ی شخصی را نمایش می‌دهد.  
در این تمرین، تمرکز روی استفاده از ساختار و تگ‌های معنایی در HTML5 است و **هیچ CSS یا JavaScript** استفاده نشده است.

---

## بخش‌های موجود در رزومه

1. **نام و اطلاعات تماس**
2. **معرفی کوتاه**
3. **تحصیلات**
4. **تجربه‌ها و پروژه‌ها**
5. **مهارت‌ها**

فایل رزومه از تگ‌های معنایی زیر استفاده می‌کند:
- `<header>`
- `<section>`
- `<article>`
- `<footer>`
